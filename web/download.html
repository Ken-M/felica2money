旧バージョン

ver 2.9 以前のバージョンはこちらからダウンロードできます。

ソースコード

最新版のソースコードは github から入手できます。 https://github.com/tmurakam/felica2money/ から入手してください。

また、ver 2.9 以前のコードは sourceforge からも入手できます。 URL は http://svn.sourceforge.jp/svnroot/moneyimport/trunk/FeliCa2Money.net/。

コンパイルには Visual Studio 2010 が必要です。
